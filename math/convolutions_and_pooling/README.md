# Convolutions and Pooling

This project implements fundamental operations in Convolutional Neural Networks (CNNs) from scratch using only **NumPy**. It covers grayscale and color image convolutions, different padding types, strides, and pooling layers.

## Learning Objectives

At the end of this project, you should be able to explain:

### General
- What is a **convolution**?
- What is **max pooling** and **average pooling**?
- What is a **kernel** / **filter**?
- What is **padding**?
- What is **"same"** padding vs **"valid"** padding?
- What is a **stride**?
- What are **channels** in convolutional layers?
- How to perform a convolution over an image
- How to perform max/average pooling over an image

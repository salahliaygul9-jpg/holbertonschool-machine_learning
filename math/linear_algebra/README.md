# Linear Algebra

Linear Algebra studies linear equations and their representations in vector spaces and matrices. It is fundamental in many areas of mathematics.  

This project covers the first part of implementing Linear Algebra concepts with Python for Machine Learning.

## Topics Covered

This project in the Machine Learning Programming series addresses:

- **Vectors**: What is a vector?
- **Matrices**: What is a matrix?
- **Transpose**: What is a transpose?
- **Matrix Shape**: What is the shape of a matrix?
- **Axis**: What is an axis?
- **Slicing**: What is a slice? How do you slice a vector/matrix?
- **Element-wise Operations**: What are element-wise operations?
- **Concatenation**: How do you concatenate vectors/matrices?
- **Dot Product**: What is the dot product?
- **Matrix Multiplication**: How is matrix multiplication done?
- **NumPy**: What is NumPy and why is it important?
- **Parallelization**: What is parallelization and why is it important?
- **Broadcasting**: What is broadcasting?
